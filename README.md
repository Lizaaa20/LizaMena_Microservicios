# NombreApellido_MicroServicios

Proyecto de microservicios con .NET 8 y Docker Compose.

## Estructura

```
/
├── ProductService/     → API de productos 
├── CustomerService/    → API de clientes  
├── OrderService/       → API de órdenes   
├── Gateway/            → API Gateway YARP  
└── docker-compose.yml
```
